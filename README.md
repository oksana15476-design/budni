# Будни — лендинг кадрового агентства

Direct-response лендинг для кадрового агентства «Будни»: массовый и точечный подбор персонала, executive search. Реализован по финальному дизайну (v4, концепция «операционная система подбора») из хэндофф-бандла Claude Design.

Статический сайт без сборки и зависимостей: семантический HTML (весь контент в разметке — хорошо для SEO), CSS и ванильный JavaScript. Подключены микроразметка JSON-LD (EmploymentAgency + FAQPage), Open Graph и шрифт Golos Text.

## Структура

```
index.html      — страница целиком (все секции, мета-теги, JSON-LD, SVG-спрайт иконок)
css/styles.css  — дизайн-система и стили всех блоков
js/main.js      — интерактив: квиз, калькулятор, FAQ-аккордеон, формы, reveal-анимации
```

## Запуск локально

Сборка не нужна — достаточно любого статического сервера:

```
python3 -m http.server 8000
# открыть http://localhost:8000
```

Или просто открыть `index.html` в браузере.

## Блоки страницы

1. Шапка с навигацией и CTA «Оценить вакансию»
2. Hero с дашбордом воронки подбора (HR-OS)
3. Боли — «Симптомы незакрытой вакансии»
4. Лидмагнит — мокап «Первичной оценки вакансии»
5. Квиз из 5 вопросов + контакты, с live-сводкой ответов
6. Услуги: массовый, точечный, executive search
7. Цепочка «что входит в стоимость» (бриф → … → гарантия)
8. Тарифы (от 45 000 ₽ / 12–15% / индивидуально)
9. Калькулятор бюджета с мини-формой заявки
10. До/после, экономика найма, процесс из 5 этапов
11. Честный фильтр («Диагностика вакансии»), гарантии, снятие возражений
12. FAQ (10 вопросов), финальная форма-диагностика, футер
13. Sticky CTA на мобильных (≤720px)

## Подключение форм

Сейчас формы (квиз, калькулятор, финальная заявка) показывают состояние «Заявка отправлена» без реальной отправки. Точка подключения — функция `submitLead()` в `js/main.js`: добавьте туда отправку в CRM, Telegram-бот или на почту.

## Деплой

Любой статический хостинг: GitHub Pages, Netlify, Vercel, обычный nginx. Достаточно отдать содержимое репозитория как есть.
